Final da Década de 1980 - Windows 1.0



Lançado em 1985, o Windows 1.0 representou uma revolução nos Sistemas Operacionais, já que os anteriores eram baseados somente em texto, este trouxe uma interface gráfica sobre o modo texto, permitindo interaçoes sem a necessidade de comando de terminal.





World Wide Web



Já em 1989, na Suíça o cientista Tim Berners-Lee projetou um sistema que permitia a interconexão de documentos por meio de hiperlinks, criando uma teia global de informações.

Berners também desenvolveu a primeira linguagem de marcação de páginas Web, o HTML, e o primeiro servidor Web e o primeiro navegador web chamado de World Wide Web, dando inicio a era WWW.
